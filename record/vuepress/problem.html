# table样式问题(element ,vxetable)

TIP

参考解决vuepress table样式污染(element ,vxetable) 样式错位 (opens new window)

在.vuepress/styles/index.styl中添加

vxe-table, .el-table   
  table
    border-collapse unset
    margin unset
    display inherit
    overflow-x unset
  tr
    border-top unset
    &:nth-child(2n)
      background-color unset
  th, td
    border unset
    padding unset
