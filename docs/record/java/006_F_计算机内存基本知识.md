### 计算机内存基本知识【扩展】

计算机中所有数据都是以二进制存储的，最小存储单元是字节（byte），一个字节是8位二进制数（1byte = 8bit），用二进制表示 `00000000` ~ `11111111`，换算成
十进制就是 `0` ~ `255`，换算成十六进制是 `00` ~ `ff`

- 二进制
  - 只有`0` 或 `1` 表示 ，满 2 进 1
- 八进制
- 十进制
  - 我常用的 `0` ~ `9`，满 10 进 1
- 十六进制
