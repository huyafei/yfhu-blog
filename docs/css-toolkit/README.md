# 开始

## 介绍

[@vensst/css-toolkit](https://github.com/vensst/css-toolkit) 是个人常用的css样式类库

## 安装

### 使用npm

```shell
npm install @vensst/css-toolkit -S
```

```js
import "@vensst/css-toolkit/index.css"

```

### 使用script

```html

<script src="https://unpkg.com/@vensst/css-toolkit/index.css"></script>
```
