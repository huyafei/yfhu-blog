# 开始

::: tip
该组件库使用 Vue 2.x 开发
:::

## npm/yarn

:::: code-group
::: code-group-item npm
```shell
npm install @vensst/vue-widget --save
```
:::
::: code-group-item yarn
```shell
yarn add @vensst/vue-widget --save
```
:::
::::

## 使用

``` js
import VueWidget from "@vensst/vue-widget";
Vue.use(VueWidget);
```
