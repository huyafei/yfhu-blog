# 开始

::: tip
该组件库使用 Vue 2.x 开发
:::

## npm/yarn

:::: code-group
::: code-group-item npm
```shell
npm install -g @vensst/cli 
```
:::
::: code-group-item yarn
```shell
yarn add -g @vensst/cli 
```
:::
::::

到这 `@vensst/cli` 脚手架工具就安装好了，使用 `vensst -v` 试试是否安装成功

```shell
vensst -v
# 2.0.0
```

## 使用

``` shell
vensst create <projectName>
```
